# Pacote de Idioma Wukchumni para Joomla! 6.x

## Informações sobre o Idioma

**Nome do Idioma:** Wukchumni (Tule-Kaweah Yokuts)  
**Código do Idioma:** wuk-US  
**Região:** Estados Unidos (Califórnia)  
**Família Linguística:** Yok-Utiana > Yokutsan > Nim > Wukchumni  
**Status:** Idioma em perigo crítico (8a - "moribundo" na escala Ethnologue)

## Sobre o Wukchumni

O Wukchumni é um dialeto da língua Tule-Kaweah Yokuts, historicamente falada pelo povo Wukchumni da bifurcação leste do rio Kaweah na Califórnia. Marie Wilcox (nascida em 1933) foi a última falante nativa da língua.

### Esforços de Revitalização

Atualmente, há importantes esforços de revitalização liderados por:
- **Marie Wilcox** e sua filha **Jennifer Malone**, que compilaram um dicionário Wukchumni
- **Destiny Treglown** (bisneta de Marie Wilcox), que está criando seu filho Oliver como falante de Wukchumni
- Programa de Aprendizagem de Idiomas para Mestres Aprendizes
- Aulas ensinadas para membros tribais interessados

## Nota Importante sobre as Traduções

⚠️ **AVISO:** Este pacote de idioma foi criado com propósitos educacionais e de preservação cultural. As traduções foram feitas com base em recursos limitados disponíveis sobre o idioma Wukchumni.

Para um pacote de idioma autêntico e preciso, seria essencial:
1. Consultar falantes nativos e especialistas no idioma Wukchumni
2. Trabalhar com a comunidade tribal Wukchumni
3. Revisar as traduções com Marie Wilcox, Jennifer Malone ou outros especialistas no idioma
4. Validar a terminologia técnica com consultores linguísticos

## Instalação

1. Faça backup do seu site Joomla
2. Acesse: Administrador → Sistema → Instalar → Extensões
3. Selecione o pacote de idioma wuk-US
4. Clique em "Upload & Instalar"
5. Vá para: Sistema → Gerenciar → Idiomas
6. Publique o idioma Wukchumni (wuk-US)

## Ativação

### Para o Site
1. Vá para: Sistema → Gerenciar → Idiomas
2. Clique na estrela ao lado de "Wukchumni (wuk-US)" para torná-lo padrão

### Para Artigos Específicos
1. Ao criar/editar um artigo
2. Selecione "Wukchumni" no campo de idioma

## Contribuições

Este é um projeto de preservação cultural. Se você tem conhecimento do idioma Wukchumni ou contato com a comunidade tribal, suas correções e melhorias são bem-vindas.
