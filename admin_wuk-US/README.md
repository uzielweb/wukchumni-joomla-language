# Pacote de Idioma Wukchumni para Joomla! Administrator

Este diretório contém o pacote de idioma **Wukchumni (wuk-US)** para a área administrativa do Joomla! 6.x.

## Estrutura do Pacote Administrator

Este pacote inclui traduções para:

### Arquivos Principais
- `joomla.ini` - Strings principais do sistema administrativo
- `lib_joomla.ini` - Biblioteca Joomla
- `langmetadata.xml` - Metadados do idioma
- `install.xml` - Arquivo de instalação
- `localise.php` - Classe PHP de localização

### Componentes Administrativos
- **com_admin.ini** - Painel administrativo
- **com_banners.ini** - Gerenciamento de banners
- **com_cache.ini** - Gerenciamento de cache
- **com_categories.ini** - Gerenciamento de categorias
- **com_config.ini** - Configurações globais
- **com_contact.ini** - Gerenciamento de contatos
- **com_content.ini** - Gerenciamento de artigos/conteúdo
- **com_cpanel.ini** - Painel de controle
- **com_fields.ini** - Campos personalizados
- **com_finder.ini** - Sistema de busca inteligente
- **com_installer.ini** - Instalador de extensões
- **com_joomlaupdate.ini** - Atualizações do Joomla
- **com_languages.ini** - Gerenciamento de idiomas
- **com_login.ini** - Login administrativo
- **com_media.ini** - Gerenciador de mídia
- **com_menus.ini** - Gerenciamento de menus
- **com_modules.ini** - Gerenciamento de módulos
- **com_plugins.ini** - Gerenciamento de plugins
- **com_privacy.ini** - Privacidade e GDPR
- **com_tags.ini** - Sistema de tags
- **com_templates.ini** - Gerenciamento de templates
- **com_users.ini** - Gerenciamento de usuários

### Template Administrativo
- **tpl_atum.ini** - Template Atum (padrão do Joomla 4/5/6)

## Instalação

O pacote de idioma administrator é instalado juntamente com o pacote do site. Para instalar:

1. Acesse: Sistema → Instalar → Extensões
2. Faça upload do pacote completo de idioma Wukchumni
3. Após instalação, ative em: Sistema → Gerenciar → Idiomas

## Ativação no Administrator

Para usar o Wukchumni no painel administrativo:

1. Vá para: Sistema → Gerenciar → Idiomas
2. Clique na aba "Installed - Administrator"
3. Clique na estrela ao lado de "Wukchumni (wuk-US)" para torná-lo padrão

Ou configure por usuário:

1. Edite o perfil do usuário
2. Na aba "Configurações Básicas"
3. Selecione "Wukchumni" em "Idioma de Backend"

## Sobre as Traduções

As traduções administrativas incluem toda a terminologia técnica necessária para gerenciar um site Joomla em Wukchumni, incluindo:
- Gerenciamento de conteúdo
- Configurações do sistema
- Administração de usuários
- Controle de extensões
- Manutenção e atualizações

⚠️ **Nota:** Para traduções precisas e autênticas, consulte especialistas no idioma Wukchumni.
